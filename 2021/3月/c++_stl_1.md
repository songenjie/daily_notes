| 六大组件   | 功能 作用                                        |
| ---------- | ------------------------------------------------ |
| contianer  | 放东西 占用内存 帮我们管理内存                   |
| allocator  | 分配器 管理 container                            |
| Algorithms | 模版函  操作容器                                 |
| iterator   | 范化的指针. Algorithms 操作 container 的桥梁     |
| Functors   | 仿函数                                           |
| Adapter    | 变压器/适配器  转换(container functors iterator) |
